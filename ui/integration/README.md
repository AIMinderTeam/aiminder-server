# 🎯 AI 목표 코칭봇 - 통합 프론트엔드

OAuth 로그인과 AI 챗봇이 통합된 SMART 목표 달성 플랫폼입니다.

## 📋 주요 기능

### 🔐 인증 시스템
- **OAuth2 로그인**: Google, Kakao 지원
- **세션 관리**: 자동 세션 확인 및 갱신
- **보안**: 쿠키 기반 인증으로 안전한 세션 관리

### 🤖 AI 챗봇
- **개인화된 목표 코칭**: SMART 목표 달성을 위한 AI 어시스턴트
- **대화 기록 관리**: 사용자별 대화 내역 저장 및 관리
- **퀵 리플라이**: 빠른 응답을 위한 버튼 인터페이스
- **실시간 타이핑 인디케이터**: 자연스러운 대화 경험

### 🎨 사용자 인터페이스
- **반응형 디자인**: 모바일/데스크톱 모두 지원
- **다크/라이트 모드**: 사용자 선호도에 따른 테마 변경
- **직관적인 네비게이션**: 쉬운 대화 기록 탐색
- **접근성**: 키보드 단축키 및 스크린 리더 지원

## 🚀 사용 방법

### 1. 서버 실행
먼저 백엔드 서버가 실행되어 있어야 합니다:
```bash
./gradlew bootRun
```

### 2. 통합 앱 접속
브라우저에서 다음 URL로 접속:
```
http://localhost:8080/ui/integration/
```

### 3. 로그인
- Google 또는 Kakao 계정으로 로그인
- 로그인 성공 시 자동으로 챗봇 화면으로 이동

### 4. 목표 코칭 시작
- AI가 환영 메시지와 함께 목표 설정을 도와줍니다
- 퀵 리플라이 버튼 또는 직접 메시지 입력으로 대화 시작

## 🎮 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl/Cmd + K` | 채팅 입력창 포커스 |
| `Ctrl/Cmd + N` | 새 대화 시작 |
| `Ctrl/Cmd + H` | 대화 기록 패널 열기/닫기 |
| `Escape` | 모달/패널 닫기 |
| `Enter` | 메시지 전송 |
| `Shift + Enter` | 줄바꿈 |

## 🔧 기술적 특징

### 모듈화된 아키텍처
```
/ui/integration/
├── index.html      # 메인 HTML 구조
├── styles.css      # 통합 CSS 스타일
├── app.js          # 메인 애플리케이션 컨트롤러
├── auth.js         # OAuth 인증 모듈
├── chatbot.js      # AI 챗봇 모듈
└── utils.js        # 공통 유틸리티 함수
```

### 이벤트 기반 통신
- **EventBus**: 모듈 간 느슨한 결합
- **실시간 상태 동기화**: 인증, 테마, 오류 상태 공유

### 로컬 스토리지 관리
- **사용자별 데이터 분리**: 다중 사용자 지원
- **자동 데이터 백업**: 로컬 스토리지 기반 대화 저장
- **데이터 내보내기**: JSON 형태로 대화 기록 내보내기

### 보안 고려사항
- **XSS 방지**: 모든 사용자 입력 이스케이프 처리
- **세션 관리**: 정기적 세션 유효성 검사
- **CSRF 보호**: 쿠키 기반 인증 사용
